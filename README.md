# 三、项目说明
Clone项目之后可直接运行，两种下拉刷新的实现区别在于：
1. **下拉刷新写在Native，然后将Weex渲染出来的View加到下拉刷新里；**
2. **封装下拉刷新为Weex的一个组件，在Weex代码中调用，灵活性更高。**

代码目录：
- components为自定义的组件；
- adapter为基础功能的接口，此处为图片下载；
- main/assets里是Weex源文件打包生成的Js文件；
- main/weex里是Weex的源文件；
